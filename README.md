# Aurora Insights

Aurora Insights is a marketing site for an intelligent operations platform that unifies data, decisions, and workflows into a single collaborative command center. The experience is built with Next.js 14 and Tailwind CSS, delivering a luminous aesthetic with interactive persona exploration, feature storytelling, momentum timeline, transparent pricing, FAQ accordion, and a guided contact form.
